🛍️ Mama Gracious Shoe Store Website

Welcome to the official website for Mama Gracious, a trusted local shoe seller offering stylish, affordable, and high-quality footwear.

📌 Project Overview

This project is a simple, responsive website built to showcase and promote Mama Gracious' shoe business online. It allows customers to browse available shoes, view prices, and easily contact the seller.

🚀 Features

🥿 Display of different shoe collections
💰 Clear pricing for each product
📱 Responsive design (works on phones & computers)
📞 Contact section for orders and inquiries
🖼️ Attractive product images
